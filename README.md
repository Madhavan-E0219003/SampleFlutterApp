# Flutter Supabase Google Auth Project

This project implements a login page for an Android app using Flutter, Supabase, and Google Sign-In.

## Prerequisites

1.  **Flutter SDK**: Ensure you have Flutter installed (`flutter doctor`).
2.  **Generate Platform Files**: Since this project was created manually, you MUST run this command in the project root to generate Android/iOS folders:
    ```bash
    flutter create .
    ```
3.  **Supabase Project**: Create a project at [supabase.com](https://supabase.com).
4.  **Google Cloud Project**: Set up a project in Google Cloud Console for OAuth.

## Setup Instructions

### 1. Configure Supabase
1.  Go to your Supabase project dashboard -> Authentication -> Providers -> Google.
2.  Enable "Google".
3.  You will need to fill in "Client ID" and "Client Secret" from Google Cloud Console, but for the *Authorized Client IDs* specifically for Android, the setup is slightly different.

### 2. Configure Google Cloud Console & Android App
1.  Create a Project on Google Cloud Console.
2.  Go to **APIs & Services > Credentials**.
3.  **Create OAuth Client ID** -> **Android**.
    - **Package name**: `com.example.can_at_door` (matches `AndroidManifest.xml` generated by `flutter create`).
    - **SHA-1 Certificate Fingerprint**: You must generate this from your local debug keystore.
      ```bash
      # Mac/Linux
      keytool -list -v -keystore ~/.android/debug.keystore -alias androiddebugkey -storepass android -keypass android
      # Windows
      keytool -list -v -keystore "\.android\debug.keystore" -alias androiddebugkey -storepass android -keypass android
      ```
    - Copy the `SHA-1` and paste it into the Google Cloud Console.
4.  **Create another OAuth Client ID** -> **Web application**.
    - This is required for the `serverClientId` in the Flutter code.
    - Copy this **Web Client ID**.
    - Also paste this Web Client ID into your Supabase Google Provider settings as the "Authorized Client ID".

### 3. Update Project Code
1.  Open `pubspec.yaml` and run `flutter pub get`.
2.  Open `lib/main.dart` and replace `YOUR_SUPABASE_URL` and `YOUR_SUPABASE_ANON_KEY` with your Supabase credentials found in Project Settings -> API.
3.  Open `lib/login_page.dart` and replace `YOUR_WEB_CLIENT_ID.apps.googleusercontent.com` with the **Web Client ID** you created in step 2.4.

### 4. Run the App
```bash
flutter run
```
